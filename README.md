# Data Cleaning Project

## Overview

This project performs data cleaning on a raw dataset by handling missing values, removing duplicates, standardizing data, detecting outliers, validating the cleaned dataset, generating reports, and creating visualizations.

## Features

- Load CSV and JSON files
- Data integrity checks
- Missing value handling
- Duplicate removal
- Data standardization
- Outlier detection
- Data validation
- Generate cleaned dataset
- Generate cleaning report
- Generate visualizations
- Logging

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
